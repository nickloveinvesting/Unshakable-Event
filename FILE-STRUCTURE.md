## File Naming Conventions

### Sections

- **Format:** `section-XX-[TYPE].md` (where XX = 01-09)
- **Examples:**
  - `section-01-DRAFT.md`
  - `section-02-VISUALS.md`
  - `section-03-INTEGRATION.md`
  - `section-04-METRICS.md`
- **Rule:** Always two-digit numbers with leading zero

### Combined Files

- **Format:** ALL CAPS with hyphens
- **Visibility:** Easily identifiable as combined files
- **Examples:**
  - `FULL-WORKBOOK-ALL-9-SECTIONS.md`
  - `BATCH-1-SECTIONS-1-2-3.md`
  - `BATCH-BUSINESS-FOCUS.md`

### Documentation

- **Format:** Title case with hyphens
- **Descriptive:** Self-explanatory names
- **Examples:**
  - `PROJECT-OVERVIEW.md`
  - `FILE-STRUCTURE.md`
  - `WORKFLOW.md`

### Research Files

- **Format:** Lowercase with hyphens
- **Descriptive:** Clear topic naming
- **Examples:**
  - `cbt-frameworks-identity-change.md`
  - `deal-analysis-frameworks.md`
  - `systems-thinking-frameworks.md`
