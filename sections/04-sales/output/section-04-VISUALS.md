# Section 4: Sales - Visual Placement Guide

## Document Purpose
This guide specifies all visual elements required for Section 4: Sales, including icons, diagrams, worksheets, and callout boxes.

---

## Section Icon

**Primary Section Icon**
- **Description:** Handshake with integrity symbol (checkmark or heart integrated)
- **Color:** Confidence Blue (#2B6CB0)
- **Placement:** Section title page, top of each major part header
- **Size:** Large on title page, small icon on part headers
- **Style:** Clean, professional, matches workbook visual system

## Diagrams & Infographics

### 1. 7 Principles of Influence Diagram
- **Location:** Part 1, after Cialdini principles explanation
- **Type:** Circular/wheel arrangement
- **Content:**
  - Center: "INFLUENCE" label
  - 7 segments radiating outward:
    1. Reciprocity (gift icon)
    2. Commitment & Consistency (checkmark icon)
    3. Social Proof (group/crowd icon)
    4. Authority (badge/certificate icon)
    5. Liking (heart/smile icon)
    6. Scarcity (hourglass/limited icon)
    7. Unity (connected people icon)
- **Colors:** Each principle has distinct color within blue/professional palette
- **Notes:** Include brief 3-word description under each principle

### 2. Archetype Sales Styles Wheel
- **Location:** Part 2, after archetype descriptions
- **Type:** 5-segment wheel with central hub
- **Content:**
  - Center hub: "SALES STYLE"
  - 5 segments:
    1. RULER - Authority-Based (crown icon)
    2. CREATOR - Innovation-Based (lightbulb icon)
    3. SAGE - Wisdom-Based (book/owl icon)
    4. WARRIOR - Results-Based (sword/target icon)
    5. CAREGIVER - Service-Based (hands/heart icon)
- **Integration:** Should visually echo Section 1's archetype wheel
- **Notes:** Each segment includes key phrase (e.g., "Leads with expertise")

### 3. Objection Handling Flow Chart
- **Location:** Part 3, after "5 Most Common Objections"
- **Type:** Decision tree/flowchart
- **Content:**
  - Start: "OBJECTION RECEIVED"
  - Step 1: ACKNOWLEDGE → "Validate their concern"
  - Step 2: CLARIFY → "Ask question to understand"
  - Step 3: RESPOND → "Address actual concern"
  - Branch: "Concern resolved?" → Yes → "Move toward close" / No → "Return to Clarify"
- **Colors:** Blue pathway, green for resolution, orange for loop-back

### 4. Sales Funnel with Conversion Rates
- **Location:** Part 5, Pipeline Management section
- **Type:** Funnel diagram (wide at top, narrow at bottom)
- **Content:**
  - Stage 1: LEAD (100) - 100%
  - Stage 2: QUALIFIED (60) - 60%
  - Stage 3: MEETING/DEMO (40) - 40%
  - Stage 4: PROPOSAL (25) - 25%
  - Stage 5: NEGOTIATION (15) - 15%
  - Stage 6: CLOSED (10) - 10%
- **Colors:** Gradient from light blue (top) to deep blue (bottom)
- **Notes:** Include "Key Metrics" sidebar with Win Rate, Cycle Length, Deal Size

### 5. Triune Brain in Sales Diagram (Optional)
- **Location:** Part 1, early in sales psychology section
- **Type:** Simplified brain cross-section
- **Content:**
  - Three labeled regions:
    1. Reptilian Brain → "Is this safe?"
    2. Limbic Brain → "Does this feel right?"
    3. Neocortex → "Does this make sense?"
- **Style:** Simple, not medical—conceptual illustration
- **Notes:** Arrow showing "Decision flows from emotional → logical"

## Worksheets

### Worksheet 4.1: Influence Principles Audit Checklist
- **Pages:** 1 full page
- **Format:** Table-based with rating columns
- **Sections:**
  - Part A: 7-principle self-assessment (rating 1-10, examples column)
  - Part B: Gap analysis (strengths, weaknesses, applications)
  - Part C: Ethics check (Yes/No checkboxes)
- **Design:** Clean table layout, adequate writing space

### Worksheet 4.2: Archetype-Based Sales Script Template
- **Pages:** 2-page spread
- **Format:** Structured template with guided prompts
- **Sections:**
  - Part A: Archetype identification
  - Part B: Script framework (Opening, Discovery, Insight, Presentation, Close)
  - Part C: 2-minute opening script writing space
  - Part D: Practice scheduling
- **Design:** Clear section headers, lined writing areas

### Worksheet 4.3: Objection Response Script Builder
- **Pages:** 2-page spread
- **Format:** Repeating template structure
- **Sections:**
  - Part A: 5-objection inventory list
  - Part B: 3-step script template (repeated 5x for each objection)
  - Part C: Practice scheduling
  - Part D: Refinement notes
- **Design:** Structured boxes for Acknowledge/Clarify/Respond pattern

### Worksheet 4.4: Closing Script Templates
- **Pages:** 1 full page
- **Format:** Checklist + writing space
- **Sections:**
  - Part A: Closing style assessment checkboxes
  - Part B: Script writing space for each close type
  - Part C: Practice commitment
  - Part D: Reflection
- **Design:** Mix of checkboxes and open writing space

### Worksheet 4.5: Sales Pipeline Tracker Template
- **Pages:** 2-page spread
- **Format:** Multiple tables and tracking grids
- **Sections:**
  - Part A: Pipeline stage definition table
  - Part B: Current pipeline audit table (large, 10+ rows)
  - Part C: Metrics dashboard table
  - Part D: Daily discipline integration
- **Design:** Structured tables with clear headers, calculation spaces
