# Real Estate Rehab Cost Estimation Reference Guide

## Overview
This comprehensive guide contains detailed rehab cost data for real estate investment properties. Base costs are Q1 2025 Dallas market rates, with location multipliers to adjust for different markets.

## How to Use This Guide
1. **Identify needed rehab items** from the categories below
2. **Select cost level** (Low/Avg/High) based on scope and conditions
3. **Apply location multiplier** based on property location
4. **Add 10-20% contingency** for unexpected issues

## Usage Notes

### Cost Selection Guidelines
- **Low**: Good existing conditions, minimal scope
- **Average**: Standard rehab scope (recommended default)
- **High**: Extensive work, poor conditions, premium materials

### Location Priority
1. Use metro-specific multiplier if available
2. Fall back to state multiplier if no metro match
3. Dallas-Fort Worth (1.0050) is the baseline market

### Key Considerations
- **Property Age**: Pre-1978 homes require hazmat testing
- **Scope Triggers**: Each item has specific conditions that trigger its need
- **Contingency**: Always add 10-20% for unexpected issues
- **Permits**: Required for structural, electrical, and plumbing work
- **Market Volatility**: Costs current as of Q1 2025

### Example Calculation
Base kitchen demo cost in Dallas: $800 (average)
San Francisco location: $800 × 1.3310 = $1,065
Detroit location: $800 × 0.8710 = $697

This reference guide enables accurate, location-adjusted rehab cost estimates for real estate investment properties across all US markets.
