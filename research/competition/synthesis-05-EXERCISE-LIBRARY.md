4. **Financial Health Snapshot** (Ramsey, Kiyosaki)

## Adaptation Recommendations for Toby's Workbook

### Recommendation 1: Crisis-Test Every Exercise
Add to each template: "How does this help you survive 2008?"

### Recommendation 2: Systems Documentation Emphasis
Even goal-setting exercises should ask: "What system ensures this happens?"

### Recommendation 3: S→B Transition Focus
Every business exercise should identify: "Does this move me from S to B quadrant?"

### Recommendation 4: Australian Directness
Simplify language, remove motivational fluff, call out excuses

### Recommendation 5: IGNITE Framework Integration
Map exercises to IGNITE letters where possible

### Recommendation 6: Artifact Creation Requirement
Every section should produce tangible document (not just internal reflection)

### Recommendation 7: Speed Optimization
Include "How fast can you complete this?" prompts

### Recommendation 8: Faith Integration (Subtle)
Where appropriate, add optional reflection: "How does this honor God/serve others?"
